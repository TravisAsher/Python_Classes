# Python Classes
This repository contains the classes that I have created in Python.

## Current Classes:
+ DnD_Character Class: Basic class for a Dungeons & Dragons character. Old project: needs updating.
+ LatLong Class: Creates an object of the type 'LatLong' that corresponds to the geographic longitude/latitude in degrees/minutes/seconds. Old project: needs updating.
+ Player Class: Creates a 'Player' class object that has various stats involved in combat and exploration. Consider this a more modern reimagining of my previous class, 'DnD_Character'.
+ Operations Class: A class used to simulate the built-in arithmetic functions of Python. Can compute basic arithmetic operators, determine primes, orders values, and more.
